# Project Management

## Overview
This folder includes all project management documents related to the Herbalife Cubing Algorithm project. It follows Agile Scrum methodology, focusing on iterative development cycles to accommodate ongoing feedback and changes.

## Contents
- **agile/**: Documentation of Agile sprints, including sprint planning, retrospectives, and sprint goals.
- **customer/**: Minutes and action items from stakeholder meetings, including feedback and project direction.
- **project_charter/**: Detailed project charter outlining objectives, scope, and deliverables.
- **timeline/**: The overall project timeline, including key milestones and deadlines.

## Current Status
Project management documents will be updated regularly to reflect sprint progress and stakeholder feedback.
