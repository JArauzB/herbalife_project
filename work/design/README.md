## Class Diagram

The **Class Diagram** represents the structure of the system, including key classes and their relationships. This section includes the following files:

- **[classDiagram.md](class_diagram/classDiagram.md)**: A detailed description of the class diagram.
- **[class_diagram.drawio](class_diagram/class_diagram.drawio)**: The diagram in draw.io format (editable).
- **[class_diagram.html](class_diagram/class_diagram.html)**: The class diagram in HTML format.
- **[class_diagram.png](class_diagram/class_diagram.png)**: The class diagram in PNG format (static view).
- **[class_diagram.svg](class_diagram/class_diagram.svg)**: The class diagram in SVG format (scalable view).
- **classDiagramNotes.txt**: Additional notes on the class diagram.

## Sequence Diagrams

The **Sequence Diagrams** describe the interaction between various components and processes in the system. The following files are included:

- **[BestFitDecreasing_Seq.drawio](sequence_diagram/BestFitDecreasing_Seq.drawio)**: Sequence diagram for the Best Fit Decreasing algorithm.
- **[PackerSequence.drawio](sequence_diagram/PackerSequence.drawio)**: Sequence diagram for the packing process.
- **[layer_result_sequence_diagram.drawio](sequence_diagram/layer_result_sequence_diagram.drawio)**: Sequence diagram for layer-node interactions.
- **[orderManagerSequence.drawio](sequence_diagram/orderManagerSequence.drawio)**: Sequence diagram for the order management process.
- **[system_sequence_diagram.drawio](sequence_diagram/system_sequence_diagram.drawio)**: System-level sequence diagram.
- **oldClassDiagram.drawio**: Older version of the class diagram for reference.
