# User stories
This part of the document will contain the user stories artifact which will show the actors, what they want to see in the visualization and why.


- <b>As a worker in the warehouse</b>, I want to see the visualization of a selected order, all this to get an overview(Box type, products, box number, product quantity) of the steps I need to make with packing process

- <b>As a worker in the warehouse</b>, I want to be able to close the visualization once I am done with the orders, all this to reset the program if I need to reset it.

- <b>As a worker in the warehouse</b>, I want to see the box in a transparent way in order to see what the inside will look like, all this to keep in track where I need to put each product

- <b>As a worker in the warehouse</b>, I want to see in the visualization a next order button to see the order of packing that I need to follow, all this to put the product in the order the algorithm decided to put the product.

- <b>As a worker in the warehouse</b>, I want to see in the visualization a previous order button to see the previous order items of the packing process, all this in case I forget where I needed to put the previous product

- <b>As a worker in the warehouse</b>, I want to see in the visualization a next box of order button to see the next box that has to be filled in, all this to put the rest of the items in the correct way for the next box

- <b>As a worker in the warehouse</b>, I want to see in the visualization a previous box of order button in case I forgot to put an item, all this to prevent any missing product in one previous box

- <b>As a worker in the warehouse</b>, I want to see in the visualization a next item of box button to see the next item that has to be included in the box, all this to complete the order 

- <b>As a worker in the warehouse</b>, I want to see in the visualization a previous item of box button in case I forgot the position of the previous item, all this to prevent any problem with the packing

- <b>As a worker in the warehouse</b>, I want to see in the visualization a next 10 items of box button to see the next 10 items that has to be included in the box, all this to complete the order 

- <b>As a worker in the warehouse</b>, I want to see in the visualization a previous 10 items of box button in case I forgot the position of the previous 10 items, all this to prevent any problem with the packing
